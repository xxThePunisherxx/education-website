
# How to start.






Clone the project

```bash
  git clone https://github.com/xxThePunisherxx/education-website
```

Go to the project directory

```bash
  cd education-website
```

Install dependencies

```bash
  npm install
```
# Run Locally

## Run Backend Server only
```bash
  npm run server
```
## Run React app only
```bash
  npm run client
```
## Run Both express server and react app
```bash
  npm run dev
```
